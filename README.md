# awesome-income

The following is a list of income strategies ranging from passive to active.

This list (and its organization) is a work in progress. Feel free to submit a PR if you have anything to add!

This list is not meant to capture traditional jobs/careers.

## Semi-Passive Income

  - Airbnb
  - Vrbo
- Monetized land
  - Collect lot rent (mobile homes, etc.)
  - [Lease to farmers](https://farmlandinfo.org/lease-your-land/)
